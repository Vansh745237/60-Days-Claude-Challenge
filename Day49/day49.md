# 🤖 Personal AI Playbook

A premium, interactive AI workflow designer that helps users create reusable AI systems instead of collecting one-off prompts.

Built as a single self-contained HTML application using **HTML, CSS, and JavaScript**, the Personal AI Playbook interviews users to understand how they work with AI and generates personalized workflows, prompt systems, and automation templates.

---

## ✨ Features

### 🎯 Personalized AI Interview

* Interactive onboarding
* Multiple-choice questions for faster setup
* Understands user role, AI experience, goals, bottlenecks, and preferred models
* Generates workflows tailored to each user

### 🧠 Personalized AI Workflow Library

Create reusable AI workflow systems instead of static prompts.

Each workflow includes:

* Editable templates
* Dynamic variables
* Practical examples
* Best practices
* One-click copy
* Favorite, duplicate, edit, and delete options

### 🏗 Prompt Builder

Build powerful prompts using reusable building blocks.

Available blocks include:

* Role
* Objective
* Context
* Constraints
* Reasoning Strategy
* Output Format
* Tone
* Examples
* Quality Checks

Features:

* Live prompt preview
* Modular prompt construction
* Easy customization
* Copy generated prompts instantly

### 🔁 Loop Builder

Convert ordinary prompts into autonomous AI workflows.

Configure:

* Goal
* Evaluation Criteria
* Improvement Strategy
* Stop Conditions
* Safety Rules

Perfect for iterative AI tasks and autonomous prompting.

### 📂 Workflow Management

* Create custom workflows
* Edit existing workflows
* Duplicate workflows
* Favorite important workflows
* Search and filter workflows
* Save automatically using Local Storage

### 📤 Import & Export

* Export workflows
* Import saved playbooks
* Backup and restore data

### 🌙 Premium User Experience

* Fully responsive design
* Dark mode
* Beautiful typography
* Smooth animations
* Keyboard shortcuts
* Interactive onboarding
* SaaS-inspired interface

### ❓ Built-in Help System

Every major feature includes clear explanations so first-time users immediately understand:

* What the tool does
* Why each section exists
* How to build reusable AI systems

## 🎯 Project Goal

Most prompt libraries become difficult to manage over time.

The Personal AI Playbook focuses on teaching users how to build **modular AI systems** that can generate thousands of prompt variations from reusable building blocks.

Instead of storing hundreds of prompts, users build scalable AI workflows that adapt to different tasks and models.
