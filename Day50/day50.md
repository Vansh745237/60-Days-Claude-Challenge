# 🚀 Day 50/60 – Claude AI Challenge

# 🛡️ Defend Your Experience

An AI-powered interview preparation platform that helps users confidently defend every claim made on their resume, portfolio, LinkedIn profile, research paper, or project documentation.

> **Day 50 of my 60 Days Claude Challenge** — Building practical AI-powered applications that solve real-world problems using modern frontend technologies and intelligent AI workflows.

Unlike traditional mock interview tools, **Defend Your Experience** acts as an intelligent skeptic. It extracts meaningful claims from uploaded documents and generates adaptive follow-up questions that become increasingly deeper and more personalized based on the user's responses.

Built as a **single self-contained HTML application** using **HTML, CSS, and JavaScript**, the app integrates with the **Anthropic Messages API** (within Anthropic's HTML artifact environment) to provide realistic, evolving interview experiences.

---

# ✨ Features

## 📄 Intelligent Document Analysis

* Upload resumes, portfolios, LinkedIn profiles, project reports, research papers, bios, and more
* Automatically extracts key achievements, skills, projects, and experience claims
* Identifies statements that require stronger evidence during interviews

## 🎤 Adaptive AI Interview

* Personalized interview generated from uploaded content
* Dynamic follow-up questions based on previous answers
* Intelligent skepticism to simulate real interview scenarios
* Conversation evolves naturally instead of following fixed question sets

## 🧠 Personalized Challenge Engine

Every claim is challenged through:

* Technical questions
* Behavioral questions
* Decision-making scenarios
* Problem-solving discussions
* Leadership and ownership validation
* Impact measurement
* Evidence-based follow-ups

## 📊 Defense Report

Receive a detailed report showing:

* Well-defended experiences
* Weak or unsupported claims
* Confidence scores
* Areas requiring stronger examples
* Suggestions for improving interview responses

## 📈 Progress Tracking

* Interview completion progress
* Confidence indicators
* Strength analysis
* Session summaries
* Performance history

## 🎨 Personalized User Interface

The application adapts to the user's preferred visual style by customizing:

* Typography
* Color palette
* Layout
* Animations
* Interaction patterns

## 💾 Local Storage Support

* Save interview sessions
* Resume later
* Store uploaded documents
* Maintain session history

## 📤 Export & History

* Export Defense Reports
* Download interview summaries
* Review previous sessions
* Track long-term improvement

## 📱 Responsive Design

* Desktop
* Tablet
* Mobile

## ⚠ Error Handling

Gracefully manages temporary API issues such as:

* Rate limits
* Network interruptions
* Temporary service failures
* Session recovery

# 🎯 Project Objective

Many candidates struggle not because they lack experience, but because they cannot confidently explain or defend the experiences listed on their resume.

This project transforms interview preparation into an adaptive conversation where every claim is questioned, validated, and strengthened—helping users build confidence before facing real interviewers.
