# 🧠 Day 45 – AI Decision Support System

## 📌 Overview

On **Day 45** of my AI learning journey, I built an **AI Decision Support System** using **Claude AI**. The application helps users make informed decisions by analyzing different options, evaluating risks, comparing outcomes, and generating AI-powered recommendations based on user inputs.

---

# 🎯 Objective

The objective of this project was to explore how AI can assist in structured decision-making by organizing information, identifying trade-offs, evaluating possible outcomes, and providing data-driven recommendations.

---
# ✨ Features

* 🧠 AI-Powered Decision Analysis
* ⚖️ Option Comparison
* 📊 Risk & Benefit Evaluation
* 🎯 Goal-Based Decision Planning
* 🔄 Scenario Analysis
* 💡 AI Recommendations
* 📈 Decision Summary Report
* 📱 Responsive User Interface

---

# 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Claude AI
* Prompt Engineering

---

# 📚 Key Learnings

* Learned how AI can simplify complex decision-making processes.
* Understood the importance of evaluating risks, benefits, and trade-offs before making decisions.
* Improved prompt engineering skills by providing structured inputs for AI analysis.
* Explored how AI-generated insights can support better planning and problem-solving.
* Strengthened frontend development skills while working with AI-generated HTML applications.

---


# 🚀 How to Run

1. Clone or download the repository.
2. Navigate to the **Day45** folder.
3. Open `ai-decision-support-system.html` in any modern web browser.
4. Enter your decision details and explore AI-generated analysis and recommendations.
