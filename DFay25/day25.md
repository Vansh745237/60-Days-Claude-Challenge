# 🦈 Day 25 – AI Shark Tank Simulator

## 📌 Overview

AI Shark Tank Simulator is a single-page HTML application that simulates a Shark Tank-style startup pitch experience. Users can enter their startup idea, answer investor questions from AI judges, receive a startup score, view an investment decision, and download a pitch report.

This project was built as part of the **Day 25 AI Challenge**.

---

## ✨ Features

- 🚀 Startup Pitch Form
- 🦈 Four AI Investor Judges
  - Venture Capitalist
  - Founder
  - Customer
  - Angel Investor
- ❓ Interactive Investor Questions
- 📊 Startup Scorecard (Out of 100)
- 💰 Investment Decision
- 📈 Suggested Valuation
- 💵 Funding Recommendation
- 🎉 Confetti Animation
- 🏆 Leaderboard (Local Storage)
- 📄 Download Pitch Report
- 📱 Fully Responsive UI
- 🌙 Modern Dark Theme

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API

# 🧪 How It Works

1. Enter your startup details.
2. Start the Shark Tank simulation.
3. Review questions from each AI judge.
4. Answer the investor questions.
5. Receive a startup score.
6. View the investment recommendation.
7. Check valuation and funding suggestions.
8. Save your score to the leaderboard.
9. Download the generated pitch report.

---

# 📈 Evaluation Criteria

The simulator evaluates startups on:

- Market Potential
- Innovation
- Business Model
- Execution
- Investment Worthiness

Final score is calculated out of **100**.
