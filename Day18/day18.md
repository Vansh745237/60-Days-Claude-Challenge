# 🧠 Brain Dump Action Planner – Claude Custom Skill

A custom Claude Skill designed to transform messy notes, meeting transcripts, brainstorming sessions, voice memos, and stream-of-consciousness thoughts into structured, actionable dashboards.

The skill automatically organizes information into summaries, action items, risks, blockers, conflicts, decisions, and open questions while preserving all original information without making assumptions.

---

## 📌 Project Overview

Managing unstructured information can be challenging when working with meetings, project discussions, brainstorming sessions, or voice notes.

The **Brain Dump Action Planner** skill helps convert raw information into a clean project-management style dashboard inspired by tools such as:

- Notion
- ClickUp
- Linear
- Asana
- Airtable

The generated output is presented as a responsive HTML dashboard with visual sections and actionable insights.

---

## ✨ Features

### 📋 Structured Summary

Automatically extracts:

- Main discussion points
- Important context
- Key decisions
- Supporting information

### 🎯 Key Takeaways

Highlights the most important points from the provided notes.

### ✅ Action Item Tracker

Displays tasks in a structured table including:

- Task
- Owner
- Deadline
- Status

### ❓ Open Questions

Captures unresolved topics and pending decisions.

### 🚨 Risks & Blockers

Identifies:

- Dependencies
- Constraints
- Risks
- Potential blockers

### ⚠️ Conflict Detection

Highlights:

- Conflicting deadlines
- Ownership conflicts
- Decision inconsistencies
- Information mismatches

### 🗂 Transcript Mode

Provides:

- Speaker Summary
- Decisions by Speaker
- Action Items by Speaker
- Attribution Notes

### 🔀 Merge Mode

Supports multiple note sources and includes:

- Duplicate Items Section
- Conflict Resolution Review
- Source Information

## 🛠 Skill Creation Process

The following steps were performed in Claude:

1. Open Claude
2. Set effort level to Low or Medium
3. Navigate to Skills section
4. Create a Custom Skill
5. Set Skill Name as **brain-dump-action-planner**
6. Add description
7. Add instructions
8. Save Skill
9. Test using notes and transcripts
10. Generate interactive dashboards
11. Review summaries and action items
12. Test multiple note formats
13. Capture screenshots
14. Upload outputs to repository
