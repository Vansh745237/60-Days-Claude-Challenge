# 📄 PDF Splitter & Merger

A modern web application developed as part of **Day 39** of my **60 Days Claude AI Challenge**. This application enables users to split large PDF documents into individual pages or custom page ranges and merge multiple PDF files into a single document—all within an intuitive, responsive interface.

---

## 📌 Project Overview

PDF Splitter & Merger is a browser-based utility that simplifies PDF management. Users can upload one or multiple PDF files, preview them, split documents by page ranges, merge multiple PDFs, and download the processed files instantly.

---

## ✨ Features

- 📂 Upload multiple PDF files
- ✂️ Split PDF by page or custom page range
- 📑 Merge multiple PDFs into one
- 👀 Preview uploaded PDFs
- 📥 Download processed PDF files
- ⚡ Fast browser-based processing
- 📊 Processing status indicators
- 🎨 Modern responsive UI
- 🌙 Clean and professional design
