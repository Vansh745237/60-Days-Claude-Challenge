# 📖 Day 27 – Prior Authorization Story Simulator

## 📌 Overview

The **Prior Authorization Story Simulator** is an interactive storytelling web application that explains the healthcare **Prior Authorization (PA)** process through the journey of **Rahul**, a patient diagnosed with Rheumatoid Arthritis.

Instead of using a traditional workflow, this simulator presents the authorization process as a story where users follow Rahul's healthcare journey, interact with different characters, make choices, and understand each stage of the insurance approval process through educational conversations. The project is implemented as a single self-contained HTML application. :contentReference[oaicite:0]{index=0}

# 🎬 Story Flow

The simulator walks users through:

1. 🩺 Doctor Visit
2. 🚧 Insurance Roadblock
3. 📚 Understanding Prior Authorization
4. 🏥 Insurance Review Process
5. ❌ Denial
6. 📨 Appeal Process
7. ✅ Approval
8. 🎓 Final Takeaways
