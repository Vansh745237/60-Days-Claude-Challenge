# Day 33 – Media Integrity Analyzer

## 🚀 Overview

On **Day 33** of my **60 Days Claude AI Challenge**, I built **Media Integrity Analyzer**, an interactive educational web application that helps users develop media literacy skills through guided learning and hands-on analysis.

Instead of testing prior knowledge, the application teaches users how to identify misleading headlines, emotional manipulation, and audience targeting techniques using interactive scenarios and beginner-friendly explanations.

---

## 🎯 Objectives

* Understand the fundamentals of media literacy.
* Learn how headlines can influence perception.
* Recognize emotional manipulation in digital content.
* Develop critical thinking when consuming online information.
* Build habits for evaluating the credibility of media.

---

## ✨ Features

* 🎨 Theme selection (including Claude Orange)
* 📰 Interactive Headline Detective challenge
* ❤️ Emotion Detector challenge
* 📊 Live Media Integrity Metrics
* 🧠 Guided explanations before every activity
* ✍️ Fair headline and neutral content rewrites
* 🤖 "How to Ask Claude" prompt engineering cards
* 📈 Final Media Integrity Dashboard
* 🔄 Replay with completely new scenarios
* 📱 Responsive editorial-style dark interface

---

## 📊 Media Integrity Metrics

* Headline Accuracy
* Source Reliability
* Emotional Manipulation
* Audience Targeting
* Overall Media Integrity Score
