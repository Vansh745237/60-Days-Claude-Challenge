# Day 37 – Task Compass: Organizational Decision Simulator

## Overview

On Day 37 of the **60 Days Claude Challenge**, I built an interactive **Task Compass** application using Claude AI. The application simulates how organizations assign responsibilities, route tasks, and manage collaboration across different workplace environments. It helps users understand organizational thinking and decision-making through practical scenarios.

## Objectives

* Learn organizational task ownership.
* Understand task routing workflows.
* Practice collaboration and decision-making.
* Analyze organizational thinking through an interactive dashboard.
* Compare decision strategies across different workplace types.

## Features

* Interactive organizational decision simulator
* Multiple workplace environments
* Task ownership identification
* Intelligent task routing workflow
* Collaboration-based decision scenarios
* Organizational Thinking Dashboard
* Replay support with different workplace types
* Fully responsive HTML application

## Technologies Used

* Claude AI
* HTML5
* CSS3
* JavaScript
