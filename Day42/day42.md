# 💰 Day 42 – Personal Financial Command Center

## 📌 Overview

On Day 42 of my AI learning journey, I built a **Personal Financial Command Center** using **Claude AI**. The application provides a centralized dashboard for managing personal finances, tracking expenses, analyzing financial health, and generating AI-powered recommendations through an interactive and responsive interface.

---

## 🎯 Objective

The objective of this project was to explore how AI can generate a complete frontend application from a structured prompt while learning the fundamentals of financial planning, budgeting, and data visualization.

---

## ✨ Features

* 📊 Financial Health Dashboard
* 💵 Income & Expense Tracker
* 📅 Budget Planning
* 💰 Savings Analysis
* 📈 Investment Overview
* 🤖 AI-Powered Financial Recommendations
* 🎯 Financial Goal Tracking
* 🔄 What-If Scenario Simulations
* 📉 Interactive Charts & Reports
* 📱 Responsive User Interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Claude AI
* Prompt Engineering

---

## 📚 Key Learnings

* Learned how AI can generate complete frontend applications from detailed prompts.
* Understood the importance of collecting user information before generating personalized financial insights.
* Explored financial health scoring and recommendation systems.
* Learned how What-If simulations help users make better financial decisions.
* Improved prompt engineering skills for AI-assisted application development.
* Strengthened frontend development knowledge using HTML, CSS, and JavaScript.
----

## 🚀 How to Run

1. Download or clone the repository.
2. Open the **Day42** folder.
3. Double-click `financial-command-center.html` or open it in your preferred web browser.
4. Explore all financial modules, AI recommendations, and What-If simulations.

---

## 🎯 Outcome

Successfully developed an AI-generated **Personal Financial Command Center** featuring financial analytics, budgeting tools, AI-driven recommendations, and interactive simulations. This project demonstrated the effectiveness of prompt engineering in rapidly building practical web applications.
