# Day 28 – Hospital Admission Readiness Simulator

## 📌 Overview

On Day 28 of my **60 Days Claude AI Challenge**, I built an interactive **Hospital Admission Readiness Simulator** using Claude AI. The application simulates the hospital admission workflow by evaluating patient readiness based on clinical, insurance, documentation, and operational factors.

The simulator helps demonstrate how different variables influence admission decisions and allows users to improve readiness through workflow actions.

---

## 🎯 Objectives

* Generate a complete HTML-based hospital admission simulator using Claude AI.
* Simulate the admission readiness process.
* Evaluate provider, physician, diagnosis, and admission details.
* Manage Prior Authorization scenarios.
* Improve readiness by resolving workflow risks.
* Analyze final admission decisions and governance metrics.

---

## 🚀 Features

* Provider and physician information entry
* Diagnosis selection
* Admission type selection
* Prior Authorization management
* Admission readiness score calculation
* Documentation risk evaluation
* Insurance verification workflow
* Clinical readiness assessment
* Bed availability tracking
* Governance Snapshot
* Final admission decision analysis
* Restart simulation with multiple patient scenarios

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Claude AI
