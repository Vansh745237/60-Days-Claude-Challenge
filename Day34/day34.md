# 🕵️ Marketing Detective

An interactive detective-style web application that transforms marketing analysis into an engaging investigation game. Instead of reading case studies, users become a marketing detective who collects evidence, analyzes campaign data, identifies strategic mistakes, and learns how successful marketing campaigns should be executed.

## 🎯 Project Overview

Marketing Detective presents randomized fictional marketing campaign cases where every investigation challenges users to identify the primary marketing mistake using campaign metrics, customer feedback, social media performance, and supporting evidence.

The application combines learning with gamification through an immersive detective experience.

## ✨ Features

* 🎨 Multiple color themes (including Claude Orange)
* 🕵️ Detective-themed premium interface
* 📂 Randomized marketing investigation cases
* 📌 Interactive investigation board
* 🗂️ Draggable evidence cards
* 🔍 Campaign metrics analysis
* 💬 Customer feedback investigation
* 📈 Marketing performance evaluation
* ✅ Case solving and verdict system
* 📚 Detailed Marketing Learning Report
* 🔄 Replay with new randomized cases
* 📱 Fully responsive design
* ⚡ Single-file HTML application

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* React (CDN) with Babel *(or Vanilla JavaScript fallback if required)*
* SVG and CSS Animations

## 🧠 Learning Outcomes

Through this project I learned:

* Identifying common marketing strategy mistakes
* Understanding campaign KPIs
* Evaluating CTR, engagement, conversions, and sales metrics
* Interpreting customer feedback and social media performance
* Building interactive educational applications
* Designing gamified learning experiences
* Creating responsive UI without external frameworks

## 🚀 How to Run

1. Download the HTML file.
2. Open it in any modern web browser.
3. Select your preferred color theme.
4. Accept a marketing investigation.
5. Organize evidence and solve the case.
6. Review the expert explanation and learning report.
7. Replay with another randomized case.
