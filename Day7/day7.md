# 💡 Key Learnings

### 1. Every Claude Model Has a Purpose

- Haiku → Speed
- Sonnet → Balance
- Opus → Depth

---

### 2. Sonnet Is the Best Daily Driver

For students, Sonnet offers:

- Fast responses
- Strong coding support
- High-quality explanations
- Excellent value

---

### 3. Standard Effort Is Usually Enough

Most tasks do not require High or Max effort.

Using Standard effort improves productivity while maintaining quality.

---

### 4. Better Context Produces Better Results

Providing:

- Goals
- Constraints
- Examples
- Desired format

significantly improves output quality.

---

### 5. Claude Can Act as a Personal AI Assistant

Useful for:

- Coding
- Learning
- Research
- Career preparation
- Resume building
- Creative projects

---

# ❌ Biggest Mistakes to Avoid

- Using Opus for simple tasks
- Writing vague prompts
- Ignoring context
- Choosing Low effort for important work
- Accepting the first response without refinement
