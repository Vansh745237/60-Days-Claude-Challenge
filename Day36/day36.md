# Day 36 – Cognitive Pattern Explorer

## 📌 Project Overview

On Day 36, I created an interactive **Cognitive Pattern Explorer** using Claude AI. This application helps users understand how their thinking patterns change under different emotional states by comparing **Calm Mode** and **Stress Mode**.

The application guides users through multiple interactive chapters, allowing them to analyze their decision-making process, complete cognitive exercises, and receive a personalized reflection journal based on their responses.

---

## 🎯 Objectives

* Generate a complete HTML application using Claude AI.
* Explore cognitive behavior in different emotional states.
* Complete interactive thinking and reflection exercises.
* Compare thinking tendencies across multiple scenarios.
* Document the project and publish it on GitHub.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Claude AI

---

## ✨ Features

* Calm Mode and Stress Mode comparison
* Interactive scenario-based activities
* Drag-and-drop cognitive exercises
* Thinking sequence arrangement
* Personalized Reflection Journal
* Thinking profile comparison
* Responsive single-page interface
