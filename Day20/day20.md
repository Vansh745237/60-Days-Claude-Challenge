# 🎮 Day 20 – AI-Powered Face Puzzle Game with Claude AI

## 📌 Project Overview

As part of Phase 3: AI-Powered App Development & Interactive Experiences, I used Claude AI to generate a fully functional Face Puzzle Game.

The application uses the browser's webcam API to capture a user's photo, automatically converts the image into puzzle pieces, and creates an interactive drag-and-drop puzzle game. The project demonstrates how modern AI can generate complete front-end applications from a single prompt.

---

## 🎯 Objectives

- Build a complete browser-based game using AI.
- Integrate webcam functionality using `getUserMedia()`.
- Capture and process user images.
- Generate puzzle pieces dynamically.
- Implement drag-and-drop gameplay.
- Support touch devices for mobile interaction.
- Track moves and completion time.
- Store scores using Local Storage.
- Explore AI-assisted front-end development.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Webcam API (`getUserMedia`)
- Drag & Drop API
- Local Storage
- Claude AI

---

## 🚀 Features

### 📷 Webcam Integration
- Accesses the user's webcam using `getUserMedia()`.
- Captures a real-time photo directly in the browser.

### 🧩 Puzzle Generation
- Automatically splits the captured image into puzzle pieces.
- Randomizes piece positions for gameplay.

### 🎮 Interactive Gameplay
- Drag-and-drop puzzle mechanics.
- Touch support for mobile devices.
- Responsive user interface.

### ⏱️ Performance Tracking
- Timer tracking.
- Move counter.
- Completion statistics.

### 💾 Score Saving
- Stores best scores locally using Local Storage.
- Persists records across browser sessions.
