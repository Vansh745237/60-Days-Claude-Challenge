# Day 12 – Job Search & Personal Branding using Claude

## Objective
The objective of this task was to optimize a resume according to a target job description and generate a personalized job search strategy using Claude AI.

---

## Tools Used
- Claude AI
- Resume (PDF/DOCX)
- Job Description (JD)
- LinkedIn / Naukri / Indeed Job Listings

---

## Steps Performed

### 1. Read Provided Resources
Reviewed all task instructions and supporting materials provided for the assignment.

### 2. Watched Solution Video
Understood the complete workflow and expected output.

### 3. Opened Claude
Accessed Claude AI and started a new conversation.

### 4. Set Effort Level to Low
Configured Claude's effort level as instructed.

### 5. Uploaded Resume
Uploaded the latest version of the resume.

### 6. Selected Target Role
Chose the desired job role based on career interests and skills.

### 7. Added Job Description
Copied a relevant Job Description from the repository (or pasted a JD from a job portal).

### 8. Added Target Companies (Optional)
Specified preferred companies for job applications.

### 9. Used Job Search & Personal Branding Prompt
Pasted the provided prompt into Claude and generated the required outputs.

### 10. Generated Deliverables
Claude produced:
- Optimized Resume
- ATS Improvement Suggestions
- Skill Gap Analysis
- Personal Branding Recommendations
- Job Search Strategy

---

## Deliverables

### Optimized Resume
- Resume tailored according to the selected Job Description.
- Improved ATS compatibility.
- Enhanced keywords and role-specific skills.

### Job Search Strategy Report
Included:
- Target roles
- Relevant skills
- ATS optimization tips
- Networking recommendations
- Personal branding improvements
- Application strategy
