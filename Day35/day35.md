# 🚀 Day 35 — 60 Days of Claude AI Challenge

## 🧩 Project: Prompt Puzzle — Master AI Prompting Through Play

Today, I built an interactive learning application focused on mastering prompt engineering through gamified challenges. The application helps users practice creating high-quality prompts using the CRAFT framework while understanding how prompt quality directly impacts AI-generated responses.

---

## 🎯 Objectives

- Learn prompt engineering through gameplay
- Practice using the CRAFT framework
- Compare weak, optimized, and over-engineered prompts
- Improve prompt writing skills with instant feedback
- Understand how better prompts produce better AI outputs

---

## ✨ Features

- Three interactive challenge modes
  - 🧩 Build the Prompt
  - 🧹 Clean the Prompt
  - ✅ Choose the Best Prompt

- Multiple randomized scenarios

- Drag-and-drop prompt construction

- Live scoring system

- Prompt Performance Report

- Prompt DNA visualization

- Personalized AI feedback

- Replay with new scenarios

- Modern responsive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Drag & Drop API

---

## 📚 What I Learned

- Importance of structured prompts
- Using the CRAFT framework effectively
- Difference between weak and optimized prompts
- Role, Context, Action, and Format in prompt engineering
- Prompt optimization techniques
- AI output quality depends heavily on prompt quality

---


## 💡 Key Takeaway

Small improvements in prompt structure can dramatically improve AI responses. A well-crafted prompt is often more valuable than simply asking a better question.
